# پوشه‌ی `data`

این پوشه فقط برای نگه‌داری فایل‌های داده استفاده می‌شود و در آن کدی قرار نمی‌گیرد.

- `data/raw/`  

  فایل اصلی دیتاست از Kaggle (مثلاً `persian_news.csv`) را در این‌جا قرار دهید.  
  به‌دلیل حجم بالا، معمولاً این فایل در مخزن Git نگهداری نمی‌شود.

- `data/processed/`  

  هر فایل تمیزشده یا زیرمجموعه‌ای را که در نوت‌بوک‌ها می‌سازید، در این پوشه ذخیره کنید  
  (مثلاً `news_subset.csv`).
